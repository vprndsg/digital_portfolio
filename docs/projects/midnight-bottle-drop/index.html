Splash Pages / Launch / Drop Page

Midnight Bottle Drop

A fast splash-page system for a timed release, balancing urgency with enough editorial framing to feel intentional.

This project turns a simple drop page into something sharper: a clear action path wrapped in mood, proof, and product context.

Presented as a working case study, with more estate-brand polish than concept-page theater.

Challenge

Timed launches can easily become generic urgency pages. The task here was to preserve speed and clarity while giving the drop a specific identity people would remember.

Deliverables

  • Pre-drop waitlist page
  • Live drop page
  • Sold-out fallback page
  • Countdown and social assets

Approach

How the project was structured.

1 / Approach

Give the page a before, during, and after state

The system plans for pre-launch waitlist mode, live-drop conversion mode, and sold-out mode so the story does not collapse when inventory changes.

2 / Approach

Balance tempo and detail

The above-the-fold area is built for a quick decision, while lower modules let more curious shoppers understand why the product matters.

3 / Approach

Make urgency feel branded

Typography, motion, and the countdown treatment support the mood instead of looking like a tacked-on commerce widget.

Artifacts

The practical parts of the work.

State design

Three launch states

Waitlist, live, and sold-out states share one visual system so the page still feels composed when the status changes.

Conversion UX

Single-decision hero

The first screen is built around one action and one reason to care, reducing hesitation without stripping away mood.

Content framing

Editorial support below the fold

Tasting notes, origin details, and serving cues support the sale instead of hiding behind generic specs.

This is the kind of page I like to build: crisp, narrow, and persuasive without feeling sterile.
